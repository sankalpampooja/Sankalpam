# Multi-Timezone Digital Clock\n\nA beautiful, responsive web application that displays the current time across multiple time zones simultaneously.\n\n## Features\n\n✨ **Key Features:**\n- 🌍 Support for 30+ time zones worldwide\n- 📱 Fully responsive design (desktop, tablet, mobile)\n- ⏰ Real-time clock updates every second\n- 🎨 Beautiful gradient UI with smooth animations\n- 🗑️ Easy timezone management (add/remove)\n- 💾 Persistent storage with localStorage\n- 🌓 Dynamic UTC offset calculation\n- ⌛ Day of week and date display\n\n## Supported Time Zones\n\n### Asia\n- India (IST) - UTC+5:30\n- Tokyo (JST) - UTC+9:00\n- Shanghai (CST) - UTC+8:00\n- Dubai (GST) - UTC+4:00\n- Hong Kong (HKT) - UTC+8:00\n- Singapore (SGT) - UTC+8:00\n- Bangkok (ICT) - UTC+7:00\n\n### Europe\n- London (GMT/BST) - UTC+0:00/+1:00\n- Paris (CET/CEST) - UTC+1:00/+2:00\n- Berlin (CET/CEST) - UTC+1:00/+2:00\n- Moscow (MSK) - UTC+3:00\n- Istanbul (EET) - UTC+3:00\n\n### Americas\n- New York (EST/EDT) - UTC-5:00/-4:00\n- Chicago (CST/CDT) - UTC-6:00/-5:00\n- Denver (MST/MDT) - UTC-7:00/-6:00\n- Los Angeles (PST/PDT) - UTC-8:00/-7:00\n- Toronto (EST/EDT) - UTC-5:00/-4:00\n- Mexico City (CST/CDT) - UTC-6:00/-5:00\n- São Paulo (BRT/BRST) - UTC-3:00/-2:00\n\n### Africa\n- Cairo (EET) - UTC+2:00\n- Johannesburg (SAST) - UTC+2:00\n- Lagos (WAT) - UTC+1:00\n\n### Oceania\n- Sydney (AEDT/AEST) - UTC+11:00/+10:00\n- Melbourne (AEDT/AEST) - UTC+11:00/+10:00\n- Auckland (NZDT/NZST) - UTC+13:00/+12:00\n\n## How to Use\n\n1. **Open the application** in your web browser\n2. **Select a timezone** from the dropdown menu\n3. **Click \"Add\"** to display the clock for that timezone\n4. **Add multiple timezones** to compare times across the world\n5. **Click \"Remove\"** on any clock card to remove it\n\n## Default Time Zones\n\nThe application comes with 4 default time zones:\n- Asia/Kolkata (India)\n- America/New_York (New York)\n- Europe/London (London)\n- Asia/Tokyo (Tokyo)\n\nThese can be modified by editing the `activeTimezones` array in `script.js`.\n\n## Technical Details\n\n### HTML Structure\n- Semantic HTML5 markup\n- Accessible form controls\n- Responsive meta tags\n\n### CSS Features\n- CSS Grid for responsive layout\n- Gradient backgrounds and effects\n- Smooth transitions and animations\n- Mobile-first design approach\n- Media queries for different screen sizes\n\n### JavaScript Functionality\n- Real-time clock updates using `setInterval`\n- Timezone conversion using `Intl.DateTimeFormat`\n- UTC offset calculation\n- LocalStorage for persistence\n- Dynamic DOM manipulation\n- Error handling\n\n## Browser Compatibility\n\n- Chrome/Chromium ✅\n- Firefox ✅\n- Safari ✅\n- Edge ✅\n- Opera ✅\n\n## Display Information\n\nEach clock card shows:\n- **Timezone Name**: The location name (e.g., \"New York\", \"Tokyo\")\n- **UTC Offset**: Current offset from UTC (e.g., \"UTC-5:00\")\n- **Digital Time**: Current time in HH:MM:SS format\n- **Date**: Current date in the specified timezone\n- **Day of Week**: Full day name (Monday, Tuesday, etc.)\n\n## Customization\n\n### Adding More Timezones\n\nEdit the `<select>` element in `index.html` to add more timezone options:\n\n```html\n<option value=\"Asia/Kolkata\">India (IST) - UTC+5:30</option>\n<option value=\"Your/Timezone\">Your City (ABBR) - UTC±X:XX</option>\n```\n\n### Changing Default Timezones\n\nEdit the `activeTimezones` array in `script.js`:\n\n```javascript\nlet activeTimezones = [\n    'Your/Timezone1',\n    'Your/Timezone2',\n    'Your/Timezone3'\n];\n```\n\n### Styling\n\nModify `style.css` to customize:\n- Color scheme (update gradient colors)\n- Font sizes and families\n- Card dimensions\n- Animation speeds\n- Responsive breakpoints\n\n## Files\n\n- `index.html` - Main HTML structure\n- `style.css` - Styling and layout\n- `script.js` - Functionality and clock logic\n\n## Tips\n\n- 💡 Use for coordinating meetings across time zones\n- 💡 Perfect for tracking market hours worldwide\n- 💡 Great for managing distributed teams\n- 💡 Useful for travelers planning activities\n- 💡 Compare time zones for international clients\n\n## Future Enhancements\n\n- 🔔 Alarm functionality\n- 🌙 Dark/Light theme toggle\n- 🔊 Sound notifications\n- 📊 Meeting time finder\n- 🌐 More timezone options\n- ⚙️ Settings panel\n\n## License\n\nFree to use and modify for personal or commercial projects.\n\n---\n\n**Enjoy your global time zone clock!** 🌍⏰\n"
